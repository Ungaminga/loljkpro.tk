TES-L Tracker Release: Usage Guide

Time to open champagne: my deck tracker is now released. Of course, times ago it’s being developed by @MarioZG, but in case of his lack of time and, as i suggest, normally conservative development politics of such software, i stand in head. From 0.6.4 to 1.0 program had many changes and i had a good long month of steady work on it. Aint reason to think i will throw this project: i got a lot of good ideas, tiny bugs to fix and optimizations, that need to impliment.

Stable Releases | Nighty Builds

Usage:

  • Download and unzip ESLTracker.zip
  • Run ESLTRacker.exe
  • Confirm DLL replace.
  • Run the game.
  • Enter a deckbuilder, try edit some deck, it will generate decks for your tracker.
  • Go play some rankeds.
  • For streaming persons: if you want to show people your deck, you’ll need add new window for OBS using Overlays->Card list for streams. You can’t handle it in other way: OBS will not correct display transparent windows generated by WINForms .NET.
  • Report bugs and send proposals here.
