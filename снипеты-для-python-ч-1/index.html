Снипеты для Python, ч. 1

Питон (вам на работе могут сказать, что его надо называть пАЙтон) – простой и удобный язык, особенно если его надо использовать в качестве скриптов-однофайловиков для различных решений. Хотя я на нём даже генераторы галерей пишу. К примеру, мне надо было обновить id в cards.json, т.к. товарищ @MarioZG использовал сгенерированные с помощью MD5 функций GUID’ы, которые имеют мало общего с реальностью

Поменять я это смог в 3 шага:

  1. Меняем язык игры в Betheda.net Launcher на английский.
  2. С помощью Navicat достаём текстовую локализацию из 
    %USERPROFILE%\AppData\LocalLow\Dire Wolf Digital\The Elder Scrolls_ Legends

    создав SQL запрос и экспортировав вывод в JSON

    SELECT "replace"(Lookup.key,"-title","") as "id", "value" as "name"
FROM Lookup
WHERE "key" LIKE "%-title"
ORDER BY
Lookup.value ASC
  3. И питоном поменяем айдишники 
    import json
from collections import OrderedDict
cards = json.loads(open("cards.json", 'r').read(), object_pairs_hook=OrderedDict)
lookup = json.loads(open("Lookup.json", 'r').read(), object_pairs_hook=OrderedDict)
for i in range(0, len(cards["Cards"])):
 for j in lookup["RECORDS"]:
     if cards["Cards"][i]["name"].lower() == j["name"].lower():
         print(cards["Cards"][i]["id"], j["id"], j["name"], cards["Cards"][i]["name"])
         cards["Cards"][i]["id"] = j["id"]
open("output.json", "w").write(json.dumps(cards, indent=4, sort_keys=False))

    К тому-же у меня окончательный вариант скрипта добавляет тип действия, который используется в трекере для расчета шанса срабатывания, к примеру пророчества:

    import json
from collections import OrderedDict
cards = json.loads(open("cards.json", 'r').read(), object_pairs_hook=OrderedDict)
lookup = json.loads(open("Lookup.json", 'r').read(), object_pairs_hook=OrderedDict)
cards["Cards"][0]["triggertype"] = "action_draw" # Abecean Navigator
for i in range(0, len(cards["Cards"])):
 if "Prophecy" in cards["Cards"][i]["text"]:
     cards["Cards"][i]["triggertype"] = "prophecy"
 elif "Ally" in cards["Cards"][i]["name"]:
     cards["Cards"][i]["triggertype"] = "ally"
 for j in lookup["RECORDS"]:
     if cards["Cards"][i]["name"].lower() == j["name"].lower():
         print(cards["Cards"][i]["id"], j["id"], j["name"], cards["Cards"][i]["name"])
         cards["Cards"][i]["id"] = j["id"]
open("output.json", "w").write(json.dumps(cards, indent=4, sort_keys=False))

    Чтобы у нас не сбивалась сортировка JSON’а надо использовать OrderedDict. Читайте так-же следующую часть!
